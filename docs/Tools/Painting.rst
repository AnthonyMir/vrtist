|
| There are multiple types of painting tools: 
| 
| The drawing tube tool draws tubes with the secondary controller. Check “Project paint on surface” for the tube to be projected on the edges of an object.
| 

.. image:: /img/PalettePainting0.png
    :align: center
    :width: 300
    :class: no-scaled-link

|
| The paint ribbon tool draws ribbons instead of tubes.
|

.. image:: /img/PalettePainting1.png
    :align: center
    :width: 300
    :class: no-scaled-link

|
| The paint hull tool creates a volume whose edges follow the controller.
| 

.. image:: /img/PalettePainting2.png
    :align: center
    :width: 300
    :class: no-scaled-link
    
|
| The paint volum tool creates a volume by making cells that will link to other close cells of the same drawing.
|
